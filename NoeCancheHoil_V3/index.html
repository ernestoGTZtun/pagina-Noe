CSS Propiedades

Las propiedades de CSS son todas aquellas características de los elementos a las que podemos aplicar estilo. Desde un color de fondo a una anchura.

Que son las Animaciones

Básicamente una animación CSS se define como el la transición gradual de unos estilos a otros en los elementos del html. Es decir, en el CSS vas a poder definir una animación mediante los estilos de partida y los estilos que quieras al final de la animación, y el navegador se va a encargar de hacer el paso de unos estilos a otros de forma gradual.

¿Qué es CSS?

CSS es un lenguaje que se utiliza para estilizar los elementos de un lenguaje de marcado como HTML. Hay 3 maneras de lidiar con esto.

¿Qué aprenderé en este curso?

¿Como funciona el css?

CSS funciona de una manera muy simple. Los navegadores Web al aplicar las reglas CSS a un documento modifican la manera en que este nos es presentado. Todo se produce de manera transparente al usuario.
